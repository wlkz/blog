2017 年终总结

wlkz 12月 31, 2017
  • 分享到微博
  • 分享到 Twitter

嗯又到了这个时候了啊，又一年了啊。写到这里的时候很自然地卡了下壳。诶今年怎么说呢，感觉发生了很多事情，然而翻了翻相册，却好像啥都没有。。好吧那就想到哪里说到哪吧。

#2017的5个心愿
1.驾照…
2.明年打比赛顺利好好刷学分最好就能公费旅游了能拿奖学金就更好了（想太多
3.多多出去玩
4.好好学习一个，希望能弄个更好玩的东东出来
5.大家开开心心，快快乐乐。
点十个人
祝认真填完的人梦想实现💓点明年的我可以吗（逃

第一点算是完事了，不说了。

先讲第三条吧。

首先是去了西安。

算是比较好玩的地方吧，博物馆真的强，吃的也很多。

接着第二点讲吧。

说实话，这条算是比较完整实现了，可能真的算是万万没想到，我这种渣渣居然也能四处公费玩了，流下心酸的泪。

北京。

回想起来上次去北京的事情，上次来算是旅游，这次来居然是比赛。

还好算是稳定输出，没空手而归。

南宁。

算是为了逃离北京的大佬而选了去南宁，为了打这个放假由姑且留了一个多月，顺便带了下新人（话说他们真的强，我校希望啊orz）。十一回来之后也每周打模拟赛，可以说是挺上心的，毕竟x万ACMer的梦想（感觉都没上万）。

去之前都和队友立起来了flag，算是保铜，毕竟模拟赛大多都是铜。

去到当地正好冷空气过境，着实复习了一下南方的湿冷，冻得一堆北方来的同学都受不了。

比赛过程就不细谈了，反正就是卡题被吊打，打铁归来。诶那时候真的心态炸了，想想该拿的都拿过，就差一个区域赛，结果不能如愿这真的难以接受。总之，没办法的啊，实力和运气都不占据只能是gg了。

这之后没没时间想其他了，all in的结果之后的六级和大作业都接连爆炸，都是凑合考，xjb写（对不起同组的人，通宵也carry不了了，被学霸和clone党吊打了），浑浑噩噩就到了结束周。

所以就到了最后一天，前年复习，去年复习，今年还tm是复习。就不知道明年是怎样了，是刚考完研呢，还是实习呢？

算了，好好复习吧，想那么多干嘛。

#2018的5个心愿
1.好好考次六级
2.明年比赛顺利
3.好好学习一个，希望能弄个更好玩的东东出来
4.希望能认识更多有趣的人
5.大家开开心心，快快乐乐。

怎么想想今年六级这样糟蹋了，明明感觉这次好像还挺简单然而没时间复习，做得一塌糊涂目测400都没。虽然说去年算是低分考过了，嘛还是想好好学下英语，裸考六级500+的大佬别笑我。

比赛呢，明年还有几个，好好打吧。至于终极目标，额看看能不能突破瓶颈吧。

其他就像字面意思，就不多说了。

感觉今年虽然像开头说得一样，好像好多事情，然而真的写只能写这些东东了。感觉就要开始面对狗日的社会和生活，考完试再想想接下来要怎么做吧，现在我脑子里都是马克思（逃

就先写那么多吧。

旧的一年，感谢有你。
新的一年，还请多多关照。
新年快乐！
